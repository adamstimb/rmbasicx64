# Release Notes

## 0.01

This is a minimal feature set pre-release.

### New features

- RM Basic-style REPL
- String-type variables
- Integer-type variables
- Float-type variables
- Variable assignment
- Expression evaluation
- RUN command
- GOTO command
- PRINT command
- LIST command
- LOAD command
- SAVE command
- EDIT command
- AUTO command
